# contador-de-cliques

#### Um contador de cliques criado em React

## Descrição

Um projeto básico de React demonstrando uma página com um contador de cliques.

Foi criado utilizando o [create-react-app](https://create-react-app.dev/) (agora obsoleto).

## Instalação

Instale o [NodeJS](https://nodejs.org/), caso já não o tenha instalado.


#### npm


1. Em seu terminal de escolha, clone o repositório com `git clone https://github.com/arthur-tsutiya/contador-de-cliques`.
2. Mude o seu diretório para a pasta do projeto com `cd contador-de-cliques`.
3. Execute `npm install` na pasta do projeto para instalar todas as dependências.
4. Com as dependências instaladas, execute `npm start` para iniciar o aplicativo em modo desenvolvedor.
5. Por padrão, o aplicativo estará disponível em `http://localhost:3000/`
